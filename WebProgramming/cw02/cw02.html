Exercise 6

I would use an external CSS Stylesheet because it reduces redundancy and makes the HTML code less dense. Editing one attribute in the CSS sheet can impact the appearance of the entire page in multiple sections, and that is something worth taking advantage of.

    Tags and Classes to Use for the Website
  • Head -- Allows for implementation of an external CSS Stylesheet, among other styling options. Also the home of the title.
  • Hyperlink: a (color:lightcoral) -- Useful for links related to the school. The color stands out, drawing users to the link.
  • Body: body (background-color:powderblue) -- This color is easy on the eyes without being too boring.
  • Headings: h1, h2 (font-weight:bold) -- Helps identify different sections of the page.
  • Paragraph: p -- Easy to use for descriptions in different sections of the page.
  • Div (border:1px outset darkgray) -- The thin border makes boundaries between sections clear, while keeping the focus on the content.
  • Table (width:60%; border:2pt solid black; background-color:floralwhite; text-align:center) -- Scheduling content is important, so it's fine if it takes up most of the width of the page. The background color is a good contrast with the border color and doesn't take away from the focus, which is the content.
  • Exam selections: list of buttons (font-family:verdana; text-align:left) -- Readability is important in exams.
  • Exam question numbers: h3 (font-weight:bold) -- It is also important to separate each question, so having a class for exam questions is helpful.
  • Departments: p (font-weight:bold) -- In menus and lists, departments are what people often look for when browsing school websites.
